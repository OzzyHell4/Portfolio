# PixelMarket — Интернет-магазин цифровых товаров

## Описание

Этот проект представляет собой полнофункциональный интернет-магазин цифровых товаров с личным кабинетом пользователя, корзиной покупок, системой поиска, поддержкой и админ-панелью.

---

## Требования

- **Open Server Panel** (рекомендуется версия 5.3.7 или новее)
- **PHP**: 7.4+
- **MySQL**: 5.7+ (или MariaDB 10.2+)
- **Браузер**: Google Chrome, Firefox, Edge, Opera

---

## Установка и запуск

1. **Скачайте и установите Open Server Panel**
   - [Скачать Open Server](https://ospanel.io/download/)
   - Установите и запустите Open Server

2. **Склонируйте или распакуйте проект в папку доменов Open Server**
   - Обычно это `C:/OSPanel/domains/kursach` или аналогичная директория

3. **Создайте базу данных**
   - Откройте phpMyAdmin (доступно по адресу http://localhost/phpmyadmin/)
   - Создайте новую базу данных, например: `pixelmarket`

4. **Импортируйте структуру базы данных**
   - В phpMyAdmin выберите созданную базу данных
   - Импортируйте файл `PixelMarket.sql` из корня проекта
   - Важно: В базе должны быть таблицы `users`, `products`, `orders`, `cart` и другие, используемые проектом

5. **Настройте подключение к базе данных**
   - Откройте файл `db_connect.php`
   - Укажите свои параметры подключения:
     ```php
     $host = 'localhost';
     $db   = 'pixelmarket';
     $user = 'root';
     $pass = '';
     ```
   - Для Open Server по умолчанию пользователь — `root`, пароль — пустой

6. **Запустите проект**
   - В Open Server добавьте домен `kursach/` или используйте http://kursach/
   - Перейдите по адресу http://kursach/ в браузере

## Структура проекта

- `index.php` — главная страница
- `catalog.php` — каталог товаров
- `product.php` — страница товара
- `cart.php` — корзина покупок
- `account.php` — личный кабинет
- `admin.php` — админ-панель
- `support.php` — система поддержки
- `search.php` — поиск товаров
- `auth.php` — авторизация
- `db_connect.php` — настройки подключения к БД
- `css/` — стили
- `js/` — JavaScript файлы
- `img/` — изображения
- `includes/` — включаемые файлы

---

## Функциональность

- Регистрация и авторизация пользователей
- Просмотр каталога товаров
- Поиск товаров
- Добавление товаров в корзину
- Оформление заказов
- Личный кабинет пользователя
- Система поддержки
- Админ-панель для управления товарами и заказами

---

## Важно
- Для работы с изображениями товаров требуется корректная настройка прав на сервере
- Для отправки email (например, для восстановления пароля) потребуется настроить почтовый сервер или SMTP
- Для полноценной работы админ-панели у пользователя должен быть статус администратора
